## 项目背景：
  地址元素识别可以抽取地址中不同的地址元素，同时也可以作为其它项目任务的基础。

## 使用:
  train_eval.py：训练与评估模型（可以选择IDCNN膨胀卷积网络或者BILSTM）  
  address_segment_service：使用Tornado部署模型（可以进行多线程部署），从而通过http协议访问服务
